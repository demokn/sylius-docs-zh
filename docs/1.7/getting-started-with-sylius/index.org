* Sylius入门
  [[https://sylius.com/online-course/][Sylius入门-在线课程（8h） 现在学习]]

  并能够在其新的网上商店中销售第一批产品。它显示了从一个想法（“我想在线销售一些东西”）
  到第一个结果（“我可以在线销售一些东西！”）的最快，最简单的方法。
